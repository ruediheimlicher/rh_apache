Recorder:
Der Recorder ist sehr einfach und kann von den Kindern auf einem mobilen Gerät selbst bedient werden. Sie können ihre Aufnahme abhören, wiederholen und schliesslich speichern.
Administrator:
Die Aufnahmen sind in Projekten organisiert und können rationell abgehört werden. Anmerkungen zu den Aufnahmen sind möglich und bei der nächsten Aufnahme sichtbar. Die Anmerkungen lassen sich ausdrucken.
Die Namen können von einer Klassenliste importiert werden.
Die Daten der Klasse können auf einem zentralen Computer oder einem Stick gespeichert werden. Das Programm kann auf verschiedenen Computern im Netzwerk gleichzeitig benutzt werden und dabei auf die Daten der Klasse zugreifen.
Die Aufnahmen lassen sich einzeln oder als Serie in verschiedenen Formaten exportieren.

SndCalc
Kopfrechnungsprogramm
Das Programm gibt eine Serie von Kopfrechnungen in Testform aus.
Es lassen sich verschiedene Optionen in Bezug auf Aufgabenart, Anzahl der Aufgaben und Zeitlimite einstellen und als eigene Tests speichern. Die Tests können den SchülerInnen individuell zugewiesen werden.
Die Ergebnisse werden gespeichert und lassen sich als Statistik auf verschiedene Arten abrufen und ausdrucken.
Die Daten der Klasse können auf einem zentralen Computer oder einem Stick gespeichert werden. Das Programm kann auf verschiedenen Computern im Netzwerk gleichzeitig benutzt werden und dabei auf die Daten der Klasse zugreifen.

  GUT 1 Rechtschreibe-Training
GUT 1 ist ein kommerzielles Rechtschreibe-Trainigsprogramm zur individuellen Förderung. Es basiert auf dem Karteikasten-System und berücksichtigt die Erkenntnisse von Piaget über die Eigenheiten des Gedächtnisses.
Das Programm GUT Start ermöglicht die einfache Installation der Schülerdateien auf einem Netzwerk-Volume und die Eingabe der Namen mit einer Klassenliste.
