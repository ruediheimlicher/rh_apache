Audio Recorder

Copyright 2002-2005 Ben Shanfelder (audiorecorder@gmail.com)

Introduction

Audio Recorder est une application scriptable qui permet l'enregistrement audio à partir de la source par défaut et l'enregistre en format AIFF, Apple Lossless, MP3, MP4 ou WAV.

Configuration requise

  • Mac OS X 10.1 ou plus récent
  • QuickTime 6.5.2 ou plus récent
  • 500 MHz G3 ou plus rapide
  • Altivec pour l'encodage MP4
  • Le paquet LAME pour l'encodage MP3

Changement dans la version 2.0

  • Ajout du support pour les formats Apple Lossless, MP4 et WAV
  • Ajout d'une option pour afficher le fichier dans le Finder après l'enregistrement
  • Ajout d'une option pour ouvrir le fichier enregistré après l'enregistrement
  • Le système ne suspendra pas son activité pendant l'enregistrement
  • Ajout d'une option avancée pour fixer la longeur maximum (en minutes) d'un enregistrement
  • Ajout d'une option avancée pour fixer le format du nom du fichier
  • Script: La propriété"next file name" utilise le format date formatters
  • L'enregistrement MP3 est supporté par le paquet LAME

Options avancées

  • FileNameFormat – Spécifie le format du nom de fichier en utilisant un date formatters.

    Pour changer ce paramètres, ouvrez le Terminal et taper:
    defaults write com.benshan.AudioRecorder FileNameFormat -string "test-%Y%m" (pour "test-200501")

    Pour réinitialiser ce paramètres à son défaut, ouvrez le Terminal et taper:
    defaults delete com.benshan.AudioRecorder FileNameFormat

  • MaxRecordingMinutes – Spécifie la longueur maximum (en minutes) de l'enregistrement.

    Pour changer ce paramètres, ouvrez le Terminal et taper:
    defaults write com.benshan.AudioRecorder MaxRecordingMinutes -int 10 (pour 10 minutes)

    Pour réinitialiser ce paramètres à son défaut, ouvrez le Terminal et taper:
    defaults delete com.benshan.AudioRecorder MaxRecordingMinutes

  • DeleteOnCancel – Efface l'enregistrement en appuyant annuler dans le panneau d'enregistrement.

    Pour changer ce paramètres, ouvrez le Terminal et taper:
    defaults write com.benshan.AudioRecorder DeleteOnCancel -bool yes

    Pour réinitialiser ce paramètres à son défaut, ouvrez le Terminal et taper:
    defaults delete com.benshan.AudioRecorder DeleteOnCancel
