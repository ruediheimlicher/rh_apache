はじめに

Audio Recorder はデフォルトの音声入力装置からの音声を録音するスクリプタブルなアプリケーションで、AIFF、Apple ロスレス、MP3、MP4 もしくは WAV ファイルに直接録音することができます。

システム条件

  • Mac OS X 10.1 以上
  • QuickTime 6.5.2 以上
  • 500 MHz G3 以上
  • MP4 エンコードには要 Altivec
  • MP3 エンコードには要 LAME フレームワーク

バージョン 2.0 での変更点

  • Apple ロスレス、MP3、MP4 そして WAV のサポートを追加
  • 録音後に出力ファイルを Finder で表示するオプションを追加
  • 録音後に出力ファイルを再生するオプションを追加
  • 録音中はスリープに入らないようにしました
  • 最大録音時間を設定する詳細オプションを追加
  • ファイル名のフォーマットを設定する詳細オプションを追加
  • スクリプト："next file name（次のファイル名）" プロパティは 日時フォーマットを使用
  • MP3 のサポートは LAME フレームワーク を使用

詳細オプション

  • FileNameFormat - 日時フォーマット を用いてファイル名のパターンを設定します。

    この設定を変更するにはターミナルを起動して次を入力します（例 "test-200501"）：
    defaults write com.benshan.AudioRecorder FileNameFormat -string "test-%Y%m"

    デフォルト設定に戻すには次を入力します：
    defaults delete com.benshan.AudioRecorder FileNameFormat

  • MaxRecordingMinutes - 最大録音時間を「分」で設定します。

    この設定を変更するにはターミナルを起動して次を入力します（例：１０分）：
    defaults write com.benshan.AudioRecorder MaxRecordingMinutes -int 10

    デフォルト設定に戻すには次を入力します：
    defaults delete com.benshan.AudioRecorder MaxRecordingMinutes

  • DeleteOnCancel - 保存パネルから「キャンセル」を選択した場合に録音ファイルを消去します。

    この設定を変更するにはターミナルを起動して次を入力します：
    defaults write com.benshan.AudioRecorder DeleteOnCancel -bool yes

    デフォルト設定に戻すには次を入力します：
    defaults delete com.benshan.AudioRecorder DeleteOnCancel
