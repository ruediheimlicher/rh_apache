Introduction

Audio Recorder is a scriptable application that records audio from the default input device and saves it directly to an AIFF, Apple Lossless, MP3, MP4, or WAV file.

Requirements

  • Mac OS X 10.1 or greater
  • QuickTime 6.5.2 or greater
  • 500 MHz G3 or faster
  • Altivec for MP4 encoding
  • The LAME framework for MP3 encoding

Changes in version 2.0

  • Added support for Apple Lossless, MP4, and WAV
  • Added option to reveal the output file in the Finder after recording
  • Added option to open the output file after recording
  • The system won't go to sleep during recording
  • Added advanced option to set the maximum number of minutes per recording
  • Added advanced option to set the file name format
  • Scripting: The "next file name" property uses date formatters
  • MP3 is supported through the LAME framework

Advanced options

  • FileNameFormat - Specify the pattern for file names using date formatters.

    To change this setting, open the terminal and enter:
    defaults write com.benshan.AudioRecorder FileNameFormat -string "test-%Y%m" (for "test-200501")

    To revert to the default setting, open the terminal and enter:
    defaults delete com.benshan.AudioRecorder FileNameFormat

  • MaxRecordingMinutes - Specify the maximum recording time in minutes.

    To change this setting, open the terminal and enter:
    defaults write com.benshan.AudioRecorder MaxRecordingMinutes -int 10 (for 10 minutes)

    To revert to the default setting, open the terminal and enter:
    defaults delete com.benshan.AudioRecorder MaxRecordingMinutes

  • DeleteOnCancel - Delete the recording when choosing Cancel from the save panel.

    To change this setting, open the terminal and enter:
    defaults write com.benshan.AudioRecorder DeleteOnCancel -bool yes

    To revert to the default setting, open the terminal and enter:
    defaults delete com.benshan.AudioRecorder DeleteOnCancel
